# MaxPreps News

PHP-based news/content website with cron, cache, sitemap, API, and template folders.

## Tech Stack

- PHP
- XAMPP-compatible web root
- Cron scripts
- Sitemap and cache utilities

## Local Setup

Place the project under an XAMPP/PHP web root and configure local settings in private config files.

## Notes

- Keep `.env`, credentials, API keys, and private config out of Git.
- Do not commit generated cache, logs, or large downloaded media.
- Validate sitemap and cron output before production use.
